**Analysis Notebook**

Built-in Jupyter notebook viewer that displays a pre-computed data analysis of all registered projects. The notebook is rendered in-browser with support for markdown cells, code cells, and rich output (images, HTML tables, text, and stream output).

**Analysis sections:**
- **Language Distribution** — bar chart of projects by primary language
- **Contributions Analysis** — top projects by commit activity with statistics
- **Skills & Categories** — breakdown of technology skills and project categories
- **Project Timeline** — line chart of projects created per year
- **Summary Statistics** — aggregate metrics across the portfolio

**Interactive features:**
- Choose analysis type from navigation tabs
- Code cells with syntax highlighting
- Output display for text, images, HTML tables, and stream content
- Fallback message when notebook file is unavailable
