## Analysis Notebook

Built-in Jupyter notebook viewer that displays a pre-computed data analysis of all registered projects. The notebook is fetched as a static `.ipynb` file and rendered in-browser.

**Analysis sections:**
- **Language Distribution** — bar chart of projects by primary language
- **Contributions Analysis** — top projects by commit activity with descriptive statistics
- **Skills & Categories** — breakdown of technology skills and project categories
- **Project Timeline** — line chart of projects created per year
- **Summary Statistics** — aggregate metrics across the portfolio

**Interactive features:**
- Choose analysis type from the navigation
- Code cells with syntax highlighting
- Output display for text, images, and HTML tables
