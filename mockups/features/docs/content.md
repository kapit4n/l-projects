**Document Content View**

The main content area of the documentation browser showing the selected document rendered as formatted markdown. Supports full GitHub-flavored markdown rendering.

**Rendering features:**
- Typographic headings (h1-h4) with proper sizing
- Paragraphs with comfortable line height
- Lists (ordered and unordered)
- Code blocks with monospace font and border
- Inline code highlighting
- Tables with bordered cells and header styling
- Links with primary color accent
- Blockquotes with left accent border
