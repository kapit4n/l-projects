**README Viewer**

Full rendered README markdown for the project, fetched from the GitHub repository. Supports all GitHub-flavored markdown including headings, code blocks with syntax highlighting, tables, images, blockquotes, and nested lists.

**Features:**
- Scrollable container for long documents
- Syntax-highlighted code blocks
- Image rendering with rounded corners
- Table support with bordered cells
- Blockquote styling with accent border
- Inline code with monospace font
