**Recent Commits**

Displays the last 5 commits fetched from the GitHub repository. Each commit entry shows the author avatar, commit message (first line), abbreviated SHA, author name, and date.

**Details shown per commit:**
- Author avatar image (or initial fallback)
- Commit message with link to full commit on GitHub
- Abbreviated SHA (7 characters) linking to GitHub
- Author name
- Committer date formatted as "Mon DD, YYYY"
- Loading skeleton while commits are being fetched
