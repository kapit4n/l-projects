**Feature Screenshots Slider**

Interactive carousel that displays feature screenshots discovered during GitHub scraping. Each slide shows an image alongside a markdown description. Supports tabbed grouping when features are organized into categories.

**Controls:**
- Previous / Next arrow buttons for manual navigation
- Dot indicators for quick jump to any slide
- Autoplay with play/pause toggle (3-second interval)
- Tab navigation when features are grouped
- Descriptions fetched from paired markdown files in the repository
