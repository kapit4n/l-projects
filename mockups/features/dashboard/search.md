## Search & Filter

Real-time search and filtering system to quickly locate projects by name, language, or description. The search bar supports instant filtering as you type, and the chip-based filters allow multi-select by language or skill.

**How it works:**
- Free-text search matches against project name and description
- Language chips filter by primary programming language
- Skill chips filter by technology tags (React, Python, Docker, etc.)
- All filters combine (AND logic) for precise results
- Clear filters with a single click
