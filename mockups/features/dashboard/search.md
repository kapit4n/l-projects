**Search & Filtering**

Real-time search bar that filters the project list as you type. Matches against project name, description, skills, and languages. Results update instantly, showing matching project cards with highlighted search terms.

**How it works:**
- Free-text search with instant results
- Matches project name, description, skills, and language
- Works together with category/skill filter chips (AND logic)
- Empty state message guides when no results match
- Clear search with a single keystroke
