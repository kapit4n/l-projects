**Hex Grid View**

Alternate visualization mode that displays projects as hexagonal tiles instead of rectangular cards. Each hexagon shows the project name and primary language, creating a dense, visually engaging layout. The hex grid is ideal for browsing large collections at a glance.

**Key aspects:**
- Projects arranged in a honeycomb pattern
- Language color coding on each hex tile
- Click a hexagon to navigate to the project detail page
- Works alongside search and filter controls
- Toggle back to card or stats view from the toolbar
