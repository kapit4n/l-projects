**Projects Dashboard**

The main landing page displaying all registered projects in a responsive card grid. Each card shows the project name, primary language badge, contribution count, size, open issues, and key skills. The header displays aggregated stats (total projects, total commits) and the toolbar provides search, filtering, view mode toggles, export, sync, and scrape controls.

**Features:**
- Search bar with real-time filtering by name, description, skills, and languages
- Multi-select filter chips for categories and skills with AND logic
- View mode toggle between card grid, hex grid, and statistics view
- Total projects and total commits counters
- Export (JSON), sort (asc/desc), sync commits, and top-10 actions
- Scrape trigger with query and limit options
- Links to archived repos, scraped repos history, and notebooks
