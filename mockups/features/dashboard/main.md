## Projects Dashboard

The main landing page displaying all registered projects in a card-based layout. Each project card shows the name, primary language, description, contribution count, and key skills.

**Features:**
- Search bar to filter projects by name or description
- Language and skill filter chips for narrowing results
- Stats bar with aggregated metrics (total projects, languages, contributions)
- Toggle between card view and hex grid view
- Quick access to archived, scraped, and notebooks pages
- "Add Project" button for manual entries
