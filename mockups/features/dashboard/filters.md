**Filter Chips**

Multi-select category and skill filter chips that narrow the project list. Active filters are visually highlighted and can be removed individually. Multiple filters combine with AND logic for precise results.

**Usage:**
- Click a chip to toggle the filter on or off
- Active chips display a filled/highlighted style
- Category filters appear first, followed by skill filters
- Filter state persists across view mode changes
- Works alongside the free-text search bar
