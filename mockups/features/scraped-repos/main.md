**Scraped Repositories**

History of all GitHub scraping sessions. Shows each scrape session with timestamp, counts of repos added and updated, and expandable details per repository.

**Tabs:**
- **Sessions** — chronological list of scrape runs with expandable details
- **All Repos** — aggregated, deduplicated list of all unique scraped repositories

**Per-session details:**
- Timestamp of when the scrape ran
- Number of repos added (new) vs updated (existing)
- Per-repo change log (Added: / Updated:)
- "Scrape Now" button to trigger a new session
